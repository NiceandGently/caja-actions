What does content the distribution ?

Caja-Actions™ distribution contains the following components:

  • Program sources

    • a Caja™ plugin, said menu, responsible for managing displayed menus and actions through Caja™ extension system;
    • a Caja™ plugin, said tracker, which manages DBus services;
    • Caja-Actions Configuration Tool is the user interface application which lets the user edit, create, delete and organize his collection of menus and actions;
    • caja-actions-new is a command-line utility for defining a new action;
    • caja-actions-run is a command-line utility for running an existing action, taking into account the current Caja™ selection;
    • caja-actions-schemas is a command-line utility which outputs to stdout the MateConf schemas of the Caja-Actions Configuration Tool; it rather targets packagers.

  • Documentation

    • the Caja-Actions™ Reference Manual (this document) rather targets developers and packagers;
    • the Caja-Actions™ Configuration Tool User's Manual is the user's manual of the Caja-Actions Configuration Tool management user interface. The distribution contains all available translations, both as HTML and PDF documents.

  • Samples

    • The sample menus and actions described in the Caja-Actions™ Configuration Tool User's Manual are included in the standard distribution.
